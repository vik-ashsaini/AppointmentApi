
﻿# Appointment Booking API (.NET 8)

A web API to book appointments for a service center.

## ✅ Features
- Book / Cancel appointments
- List available time slots
- View upcoming appointments by email
- In-memory storage with validation

## 🏃‍♂️ Run
```bash
dotnet restore
dotnet run
```
